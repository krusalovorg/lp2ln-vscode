# lp2ln-vscode

Official VS Code extension for LP2LN (Light Protocol for Layered Peer Network).

## Functionality

The extension is designed to let developers work with network resources directly from VS Code:

- Edit and manage files inside the LP2LN network.
- Work with project code and smart contracts in-network.
- Automatically mint updates on save for a faster publish flow.

## Status

This extension is currently **not actively supported**.

Active development has been temporarily paused because the team is focused on `lp2ln-core-v2`.

## What this means

- No active feature development in this repository right now.
- Bug fixes and updates may be delayed.
- Work is expected to resume after core architecture stabilization in `lp2ln-core-v2`.

## Roadmap

After `lp2ln-core-v2` reaches a stable milestone, this extension will be updated to align with the new core capabilities and APIs.
